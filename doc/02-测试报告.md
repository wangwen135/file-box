### 4.2 详细测试记录

#### 1. 登录接口

**请求：**
```bash
POST /login
Content-Type: application/x-www-form-urlencoded

username=admin&password=admin123
```

**响应：**
```
HTTP 状态码：302
Set-Cookie: token=xxx; Max-Age=2592000; Path=/; HttpOnly
Location: /index.html
```

**结果：** ✅ 登录成功，返回 302 重定向，设置 HttpOnly Cookie

---

#### 2. 获取当前用户信息

**请求：**
```bash
GET /api/auth/user
Cookie: token=xxx
```

**响应：**
```json
{
  "username": "admin",
  "role": "ADMIN",
  "storageSpaces": ["default"],
  "currentStorageSpace": "default"
}
```

**结果：** ✅ 正确返回当前用户信息
