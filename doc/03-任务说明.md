## 一、任务概述

为 File Box 项目增加深色/浅色主题切换功能，包括登录页、文件上传页和管理后台的所有界面。

---

## 二、任务要求

1. **主题切换**
   - 支持深色模式和浅色模式
   - 所有前端页面（登录页、上传页、管理后台）
   - 主题偏好保存到 localStorage
   - 主题切换按钮全局可用

2. **配色要求**
   - 配色合理，视觉舒适
   - 深色模式具有科技感
   - 浅色模式清爽简洁
   - 确保颜色对比度适中

3. **测试验证**
   - 各界面分别使用深色/浅色模式截图
   - 查看截图确保配色合理
   - 最多重复 4 次调试
   - 测试 API 接口功能

4. **文档输出**
   - 在 doc 目录下生成任务说明
   - 记录测试情况和测试结果

## 七、已知问题

### 7.1 登录功能

**问题描述：** 登录表单提交后偶尔无法正确设置 cookie

**临时解决方案：** 使用 curl 获取 token 后手动设置 cookie

**建议修复：** 检查 AuthController 中的 cookie 设置逻辑

### 7.2 主题切换按钮

**问题描述：** 某些页面加载时主题按钮显示延迟

**原因：** theme.js 加载时机与 DOM 渲染不同步

**建议修复：** 将主题应用代码移到 head 中内联执行
