文件盒子 使用手册

自托管、轻量级的文件分享与管理工具 —— 安装、使用与管理一站通

版本 2.1.0 面向普通用户 无需数据库

1认识文件盒子

它是什么，能帮你做什么。

文件盒子（File Box）是一款轻量级、自托管的文件分享与管理工具。所有文件直接保存在服务器本地磁盘上，不需要安装数据库，启动后用浏览器即可上传、下载、在线浏览和管理文件。非常适合个人、家庭或局域网内的小团队使用。

主要功能一览

上传 / 下载

拖放、选择文件、粘贴截图、上传文本，支持在线浏览。

两种浏览视图

按时间的「时间线视图」与按目录的「目录视图」自由切换。

多存储空间

把不同用途的文件分到不同空间，互不干扰。

用户与权限

管理员 / 管理者 / 普通用户三种角色，权限清晰。

匿名访问

可开放给未登录用户上传、浏览指定空间。

Web 管理后台

在浏览器里管理用户、存储空间和系统设置。

2快速上手

从解压到登录，五分钟跑起来。

第 1 步 · 启动程序

解压发布包后进入程序目录，根据操作系统运行启动脚本：

Linux / 服务器（后台运行） 
./start.sh
Windows（前台运行，关闭窗口即停止） 
双击 start.bat   —— 或在命令行执行：start.bat
关于运行方式Linux 下程序在后台运行，脚本会打印进程号（PID），需要停止时用 kill <PID> 即可；Windows 下在一个窗口中运行，关闭窗口即可停止。

第 2 步 · 在浏览器中访问

启动成功后，在浏览器中打开：

http://localhost:8888

默认端口是 8888。如果是局域网内的其他设备访问，请把 localhost 换成本机的 IP 地址。

第 3 步 · 获取并登录初始管理员账号

首次启动时，程序会自动创建配置文件、默认存储目录和管理员账号。默认管理员用户名为 admin，密码是随机生成的，并且只会在首次启动时输出一次：

  • Linux：查看 logs/filebox.log 或 logs/out.log
  • Windows：查看弹出的启动窗口中的输出
  • 用该用户名和密码登录后，立即修改密码（见下一节）。
密码只显示一次请务必记下首次启动时输出的管理员密码。如果忘了，只能通过「重置管理员密码」重新生成一个（见常见问题）。

3登录与账户

登录、匿名访问、修改密码与退出。

账号登录

在登录页输入用户名和密码，点击「登录」即可进入主页。登录状态会保持一段时间，每次操作都会自动续期，因此正常使用期间无需反复登录。

登录安全连续 5 次密码错误后，该账号会被临时锁定 15 分钟，期间无法登录。密码长度至少为 6 位。

匿名访问

如果管理员开启了匿名访问功能，登录页会出现「匿名访问」按钮。点击即可无需账号密码进入，以匿名身份浏览和上传允许匿名的存储空间。匿名身份不能删除文件、不能管理文件夹、也不能进入管理后台。

修改自己的密码

  1. 点击页面右上角的用户名展开菜单。
  2. 选择「修改密码」。
  3. 输入当前密码和新密码（新密码至少 6 位，且不能与当前密码相同）。
  4. 保存后即可使用新密码登录。

退出登录

点击右上角用户菜单中的「退出登录」即可退出当前账号。匿名用户则点击「去登录」返回登录页。

4角色与权限

谁能做什么，一张表看明白。

文件盒子里有四种身份。普通用户登录后默认是 USER 普通用户，管理员可以分配更高权限的角色。

能力ADMINMANAGERUSER匿名
上传 / 下载 / 浏览文件可以可以可以可以
在线预览（图片 / 视频 / 文本）可以可以可以可以
删除文件可以可以不可以不可以
新建 / 重命名 / 删除文件夹可以可以不可以不可以
访问管理后台可以不可以不可以不可以

简单记法普通用户（USER）只能「存」和「看」，不能删；管理者（MANAGER）多了删除和文件夹管理，但仍进不了后台；管理员（ADMIN）拥有全部权限。

5存储空间

把文件分门别类地装进不同的「盒子」。

一个存储空间相当于一个独立的文件仓库：它有名字、对应一个磁盘目录、有自己的容量上限，还可以单独设置是否允许匿名访问。你的所有上传、浏览操作，都是针对当前选中的存储空间进行的。

切换存储空间

页面左上角的存储空间切换器会列出你有权访问的所有空间（当你只有一个空间时，它只显示当前空间名）。点击切换后，页面会刷新到对应空间的文件列表。管理员自动拥有全部空间；普通用户和匿名用户只能看到被分配 / 允许的空间。

匿名访问空间

要让匿名用户能使用某个空间，需要同时满足两个条件：管理员在系统设置里开启了「匿名访问」，并且该存储空间被勾选了「允许匿名访问」。匿名登录后只能看到这类空间。

容量每个空间都有自己的容量上限（由管理员设置）。空间快满时上传会失败，请联系管理员扩容或清理文件。

6上传文件

四种方式，总有一种适合你。

所有上传都会进入当前存储空间。主页中央有一个上传区域，支持以下方式：

拖放上传

把文件从电脑直接拖到上传区域即可开始上传。

选择文件

点击上传区域或「选择文件」按钮，从弹出的窗口中选文件。

粘贴截图

截图后按 Ctrl+V（Mac 为 ⌘+V）直接粘贴，自动保存为图片。

上传文本

在文本框里输入或粘贴文字，点「上传文本」，保存为文本文件。

每次只上传一个文件无论是拖放还是选择，每次操作只会上传第一个文件。需要批量上传时，请逐个操作。

上传进度与取消

上传过程中会弹出一个进度框，显示百分比、已传大小 / 总大小、实时速度。上传成功后会自动关闭；如果想中断，点击「取消上传」即可。

上传限制

项目限制
单个文件最大2 GB（超出会提示「文件过大」）
文件名长度最多 255 字节
可用容量不能超过当前存储空间的容量上限
同名文件自动追加编号，如 报告.pdf (1)，不会覆盖原文件
上传失败的常见原因提示「文件过大」——文件超过了 2 GB；提示「空间不足」——当前存储空间已满，请联系管理员。

7浏览与查看

两种视图，随心切换。

主页上方有两个标签页，可在两种浏览方式之间随时切换：

时间线视图（默认）

按文件的修改时间分组展示，最新的在最前面。顶部有一条横向的时间轴，可以按年份 / 月份筛选文件：点击某个年份展开其月份，点击月份即筛选该时间段内的文件。列表会随着你的滚动自动加载更多。

目录视图

像电脑里的资源管理器一样，按真实的文件夹结构浏览。顶部有面包屑导航（显示当前位置，可点击返回上级），点击文件夹可进入，文件以卡片形式展示。同样支持滚动自动加载。

在线预览

点击任意文件卡片即可打开预览，不同类型有不同表现：

文件类型预览方式
图片直接显示图片内容
视频内置播放器在线播放
文本显示开头约 300 个字符的预览
其他类型显示文件名与扩展名，可下载后用本地软件打开

8管理文件与文件夹

打开、删除与文件夹整理。

打开 / 下载文件

在任一视图中点击文件卡片即可打开或预览（见上一节）。需要保存到本地时，可使用浏览器自带的「另存为」。

删除文件 ADMIN MANAGER

在文件卡片上会出现删除图标，点击后会弹出确认窗口，显示该文件所在的存储空间和相对路径，确认后即永久删除。普通用户和匿名用户看不到删除按钮。文件删除后无法恢复，请谨慎操作。

文件夹操作（仅目录视图）ADMIN MANAGER

切换到目录视图后，管理者和管理员可以整理文件夹：

  • 新建文件夹：点击右上角「＋ 新建文件夹」按钮。
  • 重命名文件夹：在文件夹卡片上点击重命名（铅笔）图标。
  • 删除文件夹：点击删除图标；删除文件夹会连同其中所有文件一起删除，同样需要二次确认。
普通用户注意USER 和匿名身份只能上传和浏览，不能删除文件，也不能新建、重命名或删除文件夹。如需整理，请联系管理者或管理员。

9管理员后台 仅 ADMIN

在浏览器里管理整套系统。

管理员登录后，点击右上角用户菜单中的「后台管理」即可进入管理后台。后台分为四个板块：

仪表盘

一眼掌握系统全貌：存储空间数量、用户数量、文件总数、已用空间，以及每个存储空间的容量使用进度条，还有系统版本、运行时间、当前活跃会话等信息。

用户管理

在此新增、编辑、删除用户，并为每个用户分配角色和可访问的存储空间：

  • 新建用户：填写用户名、密码（至少 6 位）、角色（USER / MANAGER / ADMIN）、勾选可访问的存储空间。
  • 编辑用户：可修改角色与授权空间；密码留空则保持不变。
  • 删除用户：默认的 admin 账号不可删除。
关于 ADMIN管理员会自动拥有全部存储空间的访问权限，无需逐个分配。新建存储空间时也会自动授权给所有管理员。

存储空间管理

管理各个存储空间，每个空间可设置名称、磁盘路径、容量上限、是否允许匿名访问：

  • 新建空间：指定名称（支持中文、字母、数字、下划线、短横线）、物理路径、容量、匿名开关。
  • 删除空间：只会移除配置，不会删除磁盘上的实际文件。

系统设置

  • 分享提示：开启后，上传页左上角会显示一个可复制的局域网访问地址，方便分享给别人。
  • 匿名访问：全局开关；开启后还需至少一个存储空间单独勾选「允许匿名访问」，匿名用户才能真正进入。
即时生效管理后台的保存操作会立刻写入配置并生效，无需重启程序。

10常见问题与维护

遇到问题，先看这里。

忘记管理员密码怎么办？

使用发布包自带的管理脚本重置（会生成一个新密码，并更新配置文件，配置中只保存密码哈希）：

Linux 
./manage.sh
Windows 
manage.bat

运行后选择「Reset admin password」，新密码会在当前终端中输出一次，请立即记录并用它登录、改密。

如何修改访问端口？

编辑配置文件 config/application.yml，修改后重启程序：

config/application.yml 
server:
  port: 8888   # 改成你想要的端口

文件实际存在哪里？

文件保存在各存储空间对应的磁盘目录中。默认存储空间通常位于 data/default。你可以在管理后台的「存储空间管理」里查看每个空间的实际路径，也可以直接到对应目录下访问这些文件。

上传失败怎么办？

  • 提示「文件过大」：单文件上限为 2 GB，请压缩或拆分后再传。
  • 提示「空间不足」：当前存储空间已满，联系管理员扩容或清理。
  • 其他错误：刷新页面重试，或查看 logs/filebox.log 排查。

怎么让局域网里的别人访问？

把访问地址里的 localhost 换成本机的局域网 IP（例如 http://192.168.1.10:8888）发给他们即可。开启「分享提示」后，上传页会直接显示这个地址，复制即用。

程序重启后需要重新登录吗？

是的。登录会话保存在程序内存中，重启程序后所有用户都需要重新登录。这是正常现象，并非故障。
